# mazes - a Python3 maze package

 1. Grids

    * rectangular grids
      + 4-connected
      + 6-connected
      + 8-connected

    * glued rectangular grids
      + cylindrical
      + toroidal
      + Moebius strip

    * Kuratowski grids

    * Petersen grids

    * polar grids
 
 2. Maze generation algorithms

    * grid-specific
      + lazy binary trees
      + sidewinder trees

    * grid-independent

    * tweaking
      + dead-end removal

 3. Support algorithms

    * Bellman-Ford distance and shortest path
